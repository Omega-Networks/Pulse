# Pulse: Public Utilities and Local Services Engine

Pulse is a SwiftUI-native, open-source platform for orchestrating, monitoring, and analyzing municipal infrastructure. Running locally on Apple platforms (macOS, iOS, iPadOS, visionOS, with tvOS support in development), Pulse provides a single point of control for digital infrastructure. It offers active integrations with Netbox and Zabbix, with a roadmap to support a wide range of systems, including security (e.g., Wazuh, Milestone, Gallagher), cloud orchestration (e.g., Terrakube), SCADA, SCATS, building/facility management, consents, and water management. Featuring local LLM capabilities, Pulse ensures data sovereignty and privacy while enabling secure cross-jurisdictional access to infrastructure and data.
